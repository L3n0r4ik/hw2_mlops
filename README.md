# MLops Homework 2

Датасеты предоставлены в рамках соревнования https://www.kaggle.com/competitions/teta-ml-1-2025

Продолжение ДЗ1 с использованием Kafka, Postgres и UI для удобства

## Запуск сервиса
```bash
docker compose up --build
```

Streamlit UI для данных: http://localhost:8501
Streamlit UI с табличкой результатов: http://localhost:8503
Kafka UI: http://localhost:8080
